PART 1 · 모여라딜 — 한계

실행했고, 한계를 만났습니다.

지난 분기부터 현재까지 공동구매 총 10건을 직접 팔아보며 — 막힌 지점 3가지가 분명해졌습니다.

💸
순액법 — 순마진이 안 남는다
순액법 구조라 팔아도 순마진이 거의 남지 않았습니다.
💳
PG사 조건 — 상시판매 필요
가입한 PG사는 공동구매만으론 가입 불가 — 상시판매 구조가 필요했습니다.
🧭
막막함 — 어디서부터?
다양한 방안으로 밸류업을 하고 싶지만, 어디서부터 손대야 할지 막막했습니다.
→ 팔아보니 분명해졌습니다. 제품을 더 파는 게 아니라, 구조부터 다시 짜야 한다.
PART 1 · 모여라딜 — 구조 재설계

구조를 다시 설계했습니다 — BMA 8개.

국내 판매구조를 조사해 총액법 전환·순마진 인상 경로를 찾고, BMA 8개를 구축했습니다.

🔍
국내·해외 판매구조 분석
국내 — 수수료형(순액법) vs 총액법(자체 브랜딩), 어디서 마진이 나는지 확인. 해외 — 중국 핀둬둬: 소비자가 지인 추천으로 함께 사는 그룹바이로 광고비 없이 알리바바를 추월 → 모여라딜 엠버서더 네트워크 모델로 검증.
📐
방안 발굴 → BMA 8개 구축
3PL·CPA(상시판매) 검토 · 재고 선투자는 무리 → DFP(수요 먼저 확정 후 선매입, 선금 70%+7일 뒤 잔금 30%). 이를 묶어 비즈니스 모델 아키텍처(BMA) 8개로 구축 → 재고 리스크 없이 총액법·순마진 인상 경로 확보.
🧠
마스터프롬프트 제작 — 핵심
모여라딜의 모든 것을 담은 마스터프롬프트. 전략·기획·추진·개발 속도를 크게 끌어올리고, 방향이 흔들릴 때 판단 기준이 됩니다.
📈 매출은 잠시 멈췄지만, 갈 길이 분명해졌습니다.
PART 1 · 모여라딜 — 직접 팔아보자

PART 2 · POUR스토어 — 다음 분기

만든 도구를, POUR스토어로 이식합니다.

모여라딜·영업앱에서 검증·제작한 자산을 그대로 POUR스토어에 적용·확장합니다.

🔁 모여라딜 자산 → 이식
  • 셀러 등급제 → B2B 등급제 설정
  • 마진 계산기 → 셀러 프로모션 협업 계산기
  • 경영 대시보드 손익계산 (제작 예정)
  • 맞춤 제안서 사이트(피트페이퍼형) — Claude AI가 상품 링크·상세 캡처에서 소구포인트·판매전략 자동 생성, 열람·체류 분석 + 즉시 문의 → 스토어 제안서 제작에 활용
📊 마케팅·경영 도구
  • ROAS 측정 대시보드 + 마진 계산기 — 메타광고·셀러 협상/기획에 즉시 활용
  • 경영 대시보드 — 재고 매입단가·판매단가(정가) → 총 순익·매출 예측
🤖 영업앱 고도화 (스토어용)
  • 고객 하자 현장사진 자동 분류·누적 → 데이터 자산화
  • 하자사진 필터 → AI 맞춤 솔루션·상품 추천
  • 스토어용 영업앱 개발 → B2B 고객이 소비자 응대에 활용
🚀 검증된 자산을 재사용 → 더 빠르게, 더 정교하게. 모여라딜에서 쌓은 실행·데이터가 스토어 성장을 가속합니다.
