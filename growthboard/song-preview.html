PART 1 ─────────

🔵 모여라딜

실행하며 한계를 만나, 구조부터 다시 설계하고 — 직접 팔기까지.

PART 1 · 모여라딜 — 한계

실행했고, 한계를 만났습니다.

지난 분기부터 현재까지 공동구매 총 10건을 직접 팔아보며 — 막힌 지점 3가지가 분명해졌습니다.

💸
순액법 — 순마진이 안 남는다
순액법 구조라 팔아도 순마진이 거의 남지 않았습니다.
💳
PG사 조건 — 상시판매 필요
가입한 PG사는 공동구매만으론 가입 불가 — 상시판매 구조가 필요했습니다.
🧭
막막함 — 어디서부터?
다양한 방안으로 밸류업을 하고 싶지만, 어디서부터 손대야 할지 막막했습니다.
→ 팔아보니 분명해졌습니다. 제품을 더 파는 게 아니라, 구조부터 다시 짜야 한다.
PART 1 · 모여라딜 — 구조 재설계

구조를 다시 설계했습니다 — BMA 8개.

국내 판매구조를 조사해 총액법 전환·순마진 인상 경로를 찾고, BMA 8개를 구축했습니다.

🔍
국내·해외 판매구조 분석
국내 — 수수료형(순액법) vs 총액법(자체 브랜딩), 어디서 마진이 나는지 확인. 해외 — 중국 핀둬둬: 소비자가 지인 추천으로 함께 사는 그룹바이로 광고비 없이 알리바바를 추월 → 모여라딜 엠버서더 네트워크 모델로 검증.
📐
방안 발굴 → BMA 8개 구축
3PL·CPA(상시판매) 검토 · 재고 선투자는 무리 → DFP(수요 먼저 확정 후 선매입, 선금 70%+7일 뒤 잔금 30%). 이를 묶어 비즈니스 모델 아키텍처(BMA) 8개로 구축 → 재고 리스크 없이 총액법·순마진 인상 경로 확보.
🧠
마스터프롬프트 제작 — 핵심
모여라딜의 모든 것을 담은 마스터프롬프트. 전략·기획·추진·개발 속도를 크게 끌어올리고, 방향이 흔들릴 때 판단 기준이 됩니다.
📈 매출은 잠시 멈췄지만, 갈 길이 분명해졌습니다.
PART 1 · 모여라딜 — 직접 팔아보자

이제, 직접 팔아봅니다.

3가지 채널로 동시에 — 직접 팔며 수요·전환 데이터를 쌓습니다.

💡 공급사 수요는 확보했지만, 다음 숙제는 셀러 수요 확보였습니다. 혼자 진행하다 보니 인력이 부족해 쉽지 않았고 — 그래서 '직접 팔아보자'. 인스타그램 공식계정을 활성화하고, 자사몰을 폐쇄형에서 오픈형으로 리뉴얼하려 합니다.
📦
도매꾹 상품 등록
상품 50개 등록 → 채널 신뢰도·노출 확보.
📱
내 채널로 직접 판매 — 컨텐츠 확보 + 협찬 연결
직접 공구 판매로 수요·전환 데이터 축적. 셀러 85명 제안 → 13명 컨택, 협찬·공구 조율 중. + 셀러 희망 시, 셀러채널에서 공구진행 병합.
🧩
선택형 소싱 → 맞춤 제안서 사이트
신박템 소싱 33건 → 공급사 맞춤 입점 제안서 제작. 직접 만든 맞춤 제안서 사이트(피트페이퍼형) — 템플릿에 Claude AI가 상품 링크·상세 캡처에서 소구포인트·판매전략 자동 생성 · 슬라이드별 열람·체류 분석 · 제안서 내 즉시 문의 접수.
🧪 다음 분기 — 직접 공구 10회 실험 (개당 마진 1,000원 → 최소 2,000원 구조 목표). 샘플 구매 완료 · 직접 촬영 · 셀러 협찬 병행. 빠른 실행 → 데이터 → 개선 반복으로 빅데이터를 자산화.
🚀 모여라딜은 카테고리 제한이 없습니다. 100번 실행·실패하고 AB 테스트로 더 나은 방식을 내재화하면 — 10억, 그 이상도 어렵지 않습니다.
