03 · 직접 팔아보자

이제 직접 팔아봅니다.

4개의 가설을 동시에 실험 — 하현님이 실행을 열고, 정하님이 본격 가동합니다.

EXP 01
도매꾹 상품 등록 50개
가설 — 판매 품목이 많아 보이면 신뢰성이 올라간다. 중급 브랜드 엑셀 정리 후 자사몰(카페24) 진열.
등록 50개진행 중
EXP 02
인스타그램 자체 판매
가설 — 자체 판매 시 광고·노출·이익률이 함께 오른다. 숏폼+메타광고, 인포크링크 자동화, '~~맘' 버티컬 계정 전략.
숏폼+메타버티컬 브랜드
EXP 03
신박템 소싱 → 입점 제안
가설 — 1~3만원대 신박템은 구매 욕구가 높다. 소싱 후 공급사에 입점 제안서 발송.
신박템 33건공급사 DB 109건
EXP 04
셀러 공구 제안
가설 — 협찬·공구 제안 + 촬영 가이드 제공으로 셀러가 직접 공구를 개최하게 만든다.
셀러 DB 16건제안서 제작
⚡ 실행 원칙
도매꾹으로 직접 공구 10회 실험 (개당 마진 1천원 → 최소 2천원 남는 구조 목표). 샘플 구매 완료, 직접 촬영·셀러 협찬 병행. 빠른 실행 → 데이터 → 개선 반복으로 빅데이터를 자산화합니다.
