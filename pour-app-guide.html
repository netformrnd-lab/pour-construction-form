앱 실행 & 담당자 선택
STEP 1 / 7

본인 이름 선택 후
모드를 설정하세요

앱을 열면 담당자 선택 화면이 나타납니다. 본인 이름을 탭한 후 오늘의 영업 모드(박람회 / 상시영업)를 선택합니다. 설정이 완료되면 파란 버튼을 눌러 상담을 시작합니다.

💡
처음 사용하는 경우 — 이름이 목록에 없다면 관리자에게 등록을 요청하세요.
🏢
박람회 모드는 현장 상담에 최적화되어 있습니다. 상시영업은 방문·전화 상담 시 사용합니다.
📊
홈 화면에서 오늘 상담 건수·제안서 발송 수·누적 고객 수를 실시간으로 확인할 수 있습니다.

공법 선택
STEP 2 / 7

고객 문제에 맞는
공법을 탭하세요

상단 탭(방수·도장·보수보강·토목)으로 카테고리를 필터링한 후 고객의 건물 문제에 맞는 공법 카드를 선택합니다. 2열/3열 그리드 전환도 지원합니다.

🔍
공법 선택 트리 — 지붕→방수, 외벽→도장, 지하주차장→도장/방수, 도로→토목 탭을 먼저 선택하세요.
📋
여러 공법 선택 가능 — 상담 중 여러 공법을 보여줘도 하나의 제안서에 모두 포함됩니다.

현장 하자사진 & AI 분류
STEP 3 / 7

실제 하자사진으로
고객 공감을 이끌어내세요

앱에 내장된 실제 하자사진을 보여주며 고객의 공감을 유도합니다. 고객 현장사진을 촬영하면 AI(Claude Vision)가 하자 유형·위치·심각도를 자동으로 분류합니다.

🤖
AI 분류 흐름 — 촬영 → AI 자동 분류 → 영업사원 확인 → 저장. AI가 틀렸다면 직접 수정 후 저장하세요.
😟
"맞아요, 우리 건물이랑 똑같네요" — 실제 하자사진을 보며 고객이 문제를 공감하게 합니다.
📎
촬영(📷) 또는 첨부(📎) 두 가지 방식으로 현장사진을 추가할 수 있습니다.

STEP 5 / 7

공정 탭을 클릭하며
기술력을 시각적으로 전달하세요

공정 탭을 클릭하면 시공사진·영상이 표시됩니다. 자재 특징 버튼으로 상세 모달을 열어 기술력을 시각적으로 전달하고, KTR 공인 성능 수치로 신뢰를 강화합니다.

🔬
자재 특징 버튼을 탭하면 사진·영상·GIF 모달이 열립니다. 고객에게 직접 보여주며 설명하세요.
🏆
KTR 공인 수치를 보여주면 "데이터가 있네요"라는 반응을 이끌어낼 수 있습니다.
✅
"KTR 공인 수치가 있네요" — 타사 대비 구체적 성능 데이터로 신뢰를 형성합니다.

STEP 6 / 7

✏️ 버튼으로 고객 정보를
입력하고 상담을 마무리하세요

화면 우하단 ✏️ 버튼을 탭하면 메모 시트가 열립니다. 성함·연락처·건물명을 입력하고 다음 액션을 선택한 후 "상담 마치기"를 누릅니다. Firestore에 자동 저장됩니다.

⚠️
연락처 미입력 시 SMS 발송 불가 — 반드시 연락처를 입력하세요.
✏️
텍스트 + 사진 + 명함 촬영 모두 지원합니다. 명함을 촬영하면 자동으로 정보가 입력됩니다.
☁️
저장 즉시 Firestore에 자동 저장되어 관리자 화면에서 실시간 확인 가능합니다.
🚫
견적 금액은 현장에서 절대 제시 금지 — 방문 진단 후 산출합니다.

제안서 생성 & QR 전달
STEP 7 / 7

QR코드 한 번으로
제안서를 고객 폰에 전달하세요

저장 완료 후 "📋 고객 맞춤 제안서 생성"을 누르면 상담한 모든 공법이 담긴 제안서가 생성됩니다. QR코드를 고객이 스캔하면 폰에 즉시 저장되며, SMS로도 발송할 수 있습니다.

📱
QR 스캔 — 고객이 스캔하면 폰에 저장되어 언제든 다시 열람 가능합니다. 내부 결정권자에게 공유도 쉽습니다.
📋
여러 공법 상담 시 모두 하나의 제안서에 자동으로 포함됩니다.
🔍
재열람 — POUR솔루션 홈페이지 제안서 조회 페이지에서 성함·연락처 입력으로 언제든 다시 확인 가능합니다.
