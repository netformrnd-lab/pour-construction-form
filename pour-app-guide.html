앱 실행 & 담당자 선택
STEP 1 / 7

담당자 설정 후
모드를 선택하세요

앱을 열면 담당자 설정 화면이 나타납니다. 본인 이름을 선택하고 날짜를 확인한 뒤 오늘의 영업 모드(박람회 / 상시영업)를 선택합니다. 파란 버튼을 눌러 상담을 시작합니다.

💡
처음 사용하는 경우 — 이름이 목록에 없다면 관리자에게 등록을 요청하세요.
🎪
박람회 모드는 현장 상담에 최적화되어 있습니다. 상시영업은 방문·전화 상담 시 사용합니다.
📖
홈 화면 하단 "앱 사용 매뉴얼" 버튼에서 이 가이드를 언제든 다시 볼 수 있습니다.

STEP 2 / 7

고객 건물 문제에 맞는
공법 카드를 탭하세요

상단 탭(방수·도장·보수보강·토목)으로 카테고리를 필터링한 후 고객의 건물 문제에 맞는 공법 카드를 선택합니다. 실제 시공 사진이 카드에 표시됩니다.

🔍
공법 선택 트리 — 지붕→방수, 외벽→도장, 지하주차장→도장/방수, 도로→토목 탭을 먼저 선택하세요.
📋
여러 공법 선택 가능 — 상담 중 여러 공법을 보여줘도 하나의 제안서에 모두 포함됩니다.
✓
선택된 카드에는 파란 체크 표시가 나타납니다. 다시 탭하면 선택 해제됩니다.

현장 하자사진 & AI 분류
STEP 3 / 7

실제 하자사진으로
고객 공감을 이끌어내세요

앱에 내장된 실제 하자사진을 보여주며 고객의 공감을 유도합니다. 고객 현장사진을 촬영하면 AI(Claude Vision)가 하자 유형·위치·심각도를 자동으로 분류합니다.

🤖
AI 분류 흐름 — 촬영 → AI 자동 분류 → 영업사원 확인 → 저장. AI가 틀렸다면 직접 수정 후 저장하세요.
😟
"맞아요, 우리 건물이랑 똑같네요" — 실제 하자사진을 보며 고객이 문제를 공감하게 합니다.
📄
상단 제안서 / 시공 가이드 버튼으로 QR코드를 즉시 고객에게 전달할 수 있습니다.

왜 이 문제가 생기나
콘크리트는 타설 후 수축하면서 미세균열이 생깁니다. 기존 우레탄 방수층은 이 균열을 따라가지 못해 들뜸이 발생하고, 드레인 주변 처리가 불완전하면 그 틈으로 누수가 시작됩니다. 내부 습기가 빠져나오지 못하면 방수층이 부풀어오릅니다.

원인 설명
STEP 4 / 7

원인 영상으로
문제의 본질을 이해시키세요

원인 설명 텍스트와 영상을 통해 고객이 문제의 본질을 이해하게 합니다. "단순 노후가 아닌 구조적 원인"이라는 인식 전환이 핵심입니다.

🎬
원인 영상을 고객과 함께 보면서 설명하세요. 시각적 이해가 구매 결정을 앞당깁니다.
💡
"왜 이렇게 됐는지 알겠어요" — 고객이 원인을 이해하면 해결방안에 대한 신뢰가 높아집니다.

POUR 해결방안
STEP 5 / 7

공정 탭을 클릭하며
기술력을 시각적으로 전달하세요

공정 탭을 클릭하면 시공사진·영상이 표시됩니다. 자재 특징 버튼으로 상세 모달을 열어 기술력을 시각적으로 전달하고, KTR 공인 성능 수치로 신뢰를 강화합니다.

🔬
자재 특징 버튼을 탭하면 성능 데이터 모달이 열립니다. 고객에게 직접 보여주며 설명하세요.
🏆
KTR 공인 수치를 보여주면 "데이터가 있네요"라는 반응을 이끌어낼 수 있습니다.
✅
STEP 1~6 탭을 순서대로 클릭하며 전체 시공 공정을 고객에게 설명하세요.

메모 & 고객정보 입력
STEP 6 / 7

✏️ 버튼으로 고객 정보를
입력하고 상담을 마무리하세요

화면 우하단 ✏️ 버튼을 탭하면 메모 시트가 열립니다. 성함·연락처·건물명을 입력하고 다음 액션을 선택한 후 "상담 마치기"를 누릅니다. Firestore에 자동 저장됩니다.

🗄️
고객 DB 추출 방식
원칙 · 긴급 대응 · 사후 처리
✅
원칙 — 텍스트 직접 입력
상담 후 반드시 앱 메모 폼에 텍스트로 직접 입력하세요. 텍스트로 저장된 데이터만 Firestore DB에 자동 축적되어 관리자 통계·CRM에 활용됩니다.
🖼️
긴급 시 — 그림판(메모 사진) 사용 허용
고객이 바빠 텍스트 입력이 어려운 경우, 명함·메모지를 그림판(사진 첨부)으로 촬영하여 임시 저장할 수 있습니다. 단, 이 방식은 DB에 자동 반영되지 않습니다.
⚠️
사후 처리 필수 — 당일 텍스트 전환
그림판으로 저장한 경우 당일 업무 종료 전에 사진을 보고 앱 메모 폼에 텍스트로 재입력하여 DB를 축적하세요. 미처리 시 고객 데이터가 누락됩니다.

연락처 미입력 시 SMS 발송 불가 — 반드시 연락처를 입력하세요.

텍스트 + 사진 + 명함 촬영 모두 지원합니다. 명함을 촬영하면 자동으로 정보가 입력됩니다.

제안서 생성 & QR 전달
STEP 7 / 7

QR코드 한 번으로
제안서를 고객 폰에 전달하세요

저장 완료 후 "📋 고객 맞춤 제안서 생성"을 누르면 상담한 모든 공법이 담긴 제안서가 생성됩니다. QR코드를 고객이 스캔하면 폰에 즉시 저장되며, SMS로도 발송할 수 있습니다.

📱
QR 스캔 — 고객이 스캔하면 폰에 저장되어 언제든 다시 열람 가능합니다. 내부 결정권자에게 공유도 쉽습니다.
📋
여러 공법 상담 시 모두 하나의 제안서에 자동으로 포함됩니다.
🔍
재열람 — POUR솔루션 홈페이지 제안서 조회 페이지에서 성함·연락처 입력으로 언제든 다시 확인 가능합니다.
