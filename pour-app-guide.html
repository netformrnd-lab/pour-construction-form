① 영업사원 상담 파이프라인
앱 실행부터 제안서 전달까지 7단계
1
앱 실행 & 담당자·모드 선택
담당자 이름 선택 후 박람회/상시영업 모드를 설정합니다. 미설정 시 자동으로 선택 화면이 표시됩니다.
2
공법 선택
방수·도장·보수보강·토목 탭으로 필터링 후 고객 문제에 맞는 공법을 선택합니다.
💡 2열/3열 그리드 전환 지원
3
STEP 1 — 현장 하자사진
실제 현장 하자사진을 보여주며 공감을 끌어냅니다. 고객 현장사진을 촬영하면 AI가 하자 유형·위치·심각도를 자동 분류합니다. 확인 후 저장.
📷 촬영 → AI 분류 → 영업사원 확인 → 저장
4
STEP 2 — 왜 이 문제가 생기나
원인 설명과 원인 영상으로 고객이 문제의 본질을 이해하게 합니다. "단순 노후가 아닌 구조적 원인"이라는 인식 전환이 핵심.
5
STEP 3 — POUR 해결방안
공정 탭을 클릭하면 시공사진·영상이 표시됩니다. 자재 특징 버튼으로 상세 모달을 열어 기술력을 시각적으로 전달합니다. KTR 공인 성능 수치로 신뢰를 강화합니다.
🔬 자재 특징 버튼 → 사진/영상/GIF 모달
6
영업 메모 작성 & 저장
메모 버튼(✏️)으로 고객 정보·메모·다음 액션을 입력하고 저장합니다. Firestore에 자동 저장되어 관리자 화면에서 실시간 확인 가능합니다.
✏️ 텍스트 + 사진 + 명함 촬영 지원
7
고객 맞춤 제안서 생성 & QR 전달
저장 완료 후 "📋 고객 맞춤 제안서 생성"을 누르면 상담한 모든 공법이 담긴 제안서가 생성됩니다. QR코드로 고객 폰에 즉시 전달하거나 SMS로 발송합니다.
📱 QR 스캔 → 고객이 언제든 다시 열람 가능
② 고객이 경험하는 흐름
상담 현장에서 고객 관점의 여정
😟
1. 공감
"맞아요, 우리 건물이랑 똑같네요"
실제 하자사진을 보며 문제 공감.
💡
2. 이해
"왜 이렇게 됐는지 알겠어요"
원인 영상으로 구조적 원인 이해.
✅
3. 신뢰
"KTR 공인 수치가 있네요"
타사 대비 구체적 성능 데이터 확인.
📱
4. 수령
QR 스캔 한 번으로 제안서를 폰에 저장. 내부 결정권자에게 공유 가능.
🔍
5. 재열람 — 언제든 다시 확인
POUR솔루션 홈페이지 제안서 조회 페이지에서 성함·연락처를 입력하면 상담 내용이 담긴 제안서를 언제든 다시 확인할 수 있습니다.
③ 핵심 기능 6가지
🤖
AI 하자 자동 분류
사진 촬영 시 Claude Vision이 유형·위치·심각도를 자동 분류.
🔬
자재 특징 모달
공정별 자재의 사진·영상·GIF를 모달로 바로 보여줌.
📋
고객 맞춤 제안서
상담한 모든 공법을 담은 제안서를 자동 생성.
📱
QR 즉시 전달
제안서 링크를 QR코드로 고객 폰에 즉시 전달.
☁️
실시간 DB 저장
모든 상담은 Firebase에 즉시 저장. 관리자 실시간 확인.
📊
어드민 관리센터
통계·유입 관리·담당자·AI 설정 등 관리자 전용 기능.

⑤ 영업사원 사용 매뉴얼
처음 사용하는 영업사원을 위한 7단계
📖 POUR솔루션 앱 — 7단계 매뉴얼
이 순서대로만 하면 됩니다
1
앱 열기 → 담당자 선택
본인 이름을 선택합니다. 처음이라면 관리자에게 등록을 요청하세요.
박람회 / 상시영업 모드 선택 필수
2
상담 시작하기 클릭
홈 화면의 큰 파란 버튼을 누르면 공법 목록으로 이동합니다.
3
고객 문제에 맞는 공법 선택
상단 탭(방수·도장·보수보강·토목)으로 필터링 후 해당 공법을 탭합니다.
4
공법 화면을 스크롤하며 설명
하자사진 → 원인 설명 → 원인 영상 → 해결방안(공정 탭 클릭) → 성능 수치 순서로 고객에게 보여줍니다.
5
고객 현장사진 촬영 (선택)
하자사진 섹션에서 📷촬영 또는 📎첨부로 현장사진을 추가합니다. AI가 자동 분류하면 확인 후 저장합니다.
AI 분류가 틀렸다면 직접 수정 후 저장
6
✏️ 메모 버튼 → 고객정보 입력 → 상담 마치기
화면 우하단 ✏️ 버튼 → 성함·연락처·건물명 입력 → 메모 작성 → 다음 액션 선택 → "상담 마치기"
⚠️ 연락처 없으면 SMS 발송 불가
7
📋 제안서 생성 → QR로 전달
저장 완료 화면에서 "📋 고객 맞춤 제안서 생성"을 누릅니다. QR코드를 고객이 스캔하면 폰에 저장됩니다.
여러 공법 설명 시 모두 제안서에 포함
⚠️ 꼭 기억하세요
  • 견적 금액은 절대 현장에서 제시하지 마세요 (방문 진단 후 산출)
  • 고객 정보는 앱 메모 폼으로만 수집하세요
  • 상담 중 여러 공법을 보여줘도 하나의 제안서에 모두 포함됩니다
  • 배터리와 인터넷 연결을 항상 확인하세요
