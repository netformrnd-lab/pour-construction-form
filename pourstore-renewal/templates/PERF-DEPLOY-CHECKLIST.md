## A-2. 카페24 관리자 — 상품 진열항목 켜기 (카드 '최대 노출'의 핵심)
> CSS는 카페24가 '내보낸' 항목만 예쁘게 정렬합니다. 카드에 정보를 더 보이게 하려면
> 각 진열 모듈에서 아래 항목을 **ON** 하세요. (쇼핑몰설정 > 상품 진열/디자인 > 진열항목 설정, 또는 스마트디자인 진열영역)

각 진열영역(메인 진열·분류별 목록·검색결과·추천/신상품/베스트·관련상품·최근본상품)에서:
- [ ] **상품 요약설명(간략설명)** 표시 ON → 상품명 아래 회색 한 줄 (예: "셀프시공 OK · 옥상 누수 보수")
- [ ] **판매가 + 소비자가(정가)** 함께 표시 → 정가는 자동 취소선, 할인가 검정 볼드
- [ ] **할인기간 할인가 / 할인율(%)** 표시 ON → 오렌지 포인트로 노출
- [ ] **적립금** 표시 ON → 가격 아래 작은 회색 글씨
- [ ] **아이콘(신상품/추천/품절 등)** 표시 ON → 이름/가격 아래 정렬
- [ ] 해시태그를 쓰려면: 상품 **요약설명**에 `셀프OK 방수 옥상`처럼 키워드 입력(분류 목록 카드에 `#태그`로 자동 변환)

> 팁: 모듈마다 항목 설정이 따로입니다. 메인/분류/검색 각각 동일하게 켜야 자사몰 전체가 같은 정보량으로 보입니다.

## 확인 방법
- 저장 후 **시크릿창**으로 열기 (브라우저 캐시 회피)
- 콘솔(F12)에 `[psm1] 로드`, `[psy3]`, `[psg3]`, `[pstory]`, `[psg4]`, `[ppr2~4]` 로그 → 정상
- 메인 → POUR이야기/검색 이동 시 두 번째 화면이 거의 즉시 뜨면 캐시 정상
