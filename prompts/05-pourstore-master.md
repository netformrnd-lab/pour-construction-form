## §1. 브랜드 정의

### 1-1. 한 줄 정의
- **포지셔닝**: 쉽고 오래가는 건축물 유지보수 자재의 모든 것
- **브랜드 역할**: 실행 편의 · 구매 전환 · 셀프시공의 진입장벽 제거
- **상위 브랜드 관계**:
  - **POUR공법(기술)** 에서 검증한 자재를
  - **POUR솔루션(서비스)** 이 추천하는 그 자재 그대로
  - **누구나 살 수 있게 만드는 채널** 이 POUR스토어
- **타겟**: 셀프시공 개인 · 소규모 시공업체 · 건물관리자 · 파트너사(자재 유통/시공 매칭)
- **공식 도메인**: https://www.pourstore.net

### 1-2. 해결하는 심리적 장벽
> "어떤 자재를 사야 하는지 모르겠다. 실패하면 어쩌지?"

선택의 어려움 · 시공 실패 두려움 · 전문 용어의 벽.
→ POUR스토어는 **"무엇을 사야 하는지"** 와 **"어떻게 시공하는지"** 를 함께 해결하는 채널이다.

### 1-3. 핵심 약속 (브랜드 컨트랙트)
1. **검증된 자재만 판다** — POUR공법 시방서에 등재된 자재만 취급
2. **실패하지 않게 판다** — 단품이 아니라 "옥상 누수 패키지" 같은 문제 해결 묶음 우선
3. **시공가이드를 함께 판다** — 영상·사진·체크리스트로 셀프시공 성공률 보장
4. **소포장도 판다** — 1세대용 소량부터 단지 단위 대량까지 같은 품질
5. **결제·정산은 본사가 책임진다** — 파트너사도 카드/계좌를 직접 받지 않음 (월말 자동 수수료 정산)

## §2. 비주얼 아이덴티티 (VI)

### 2-1. 컬러 시스템
```css
:root {
  /* Primary — 오렌지 (실행/구매 유도) */
  --or:       #F25C05;        /* POUR 시그니처 오렌지 (브랜드 메모리) */
  --or2:      #F49A3A;
  --or-pale:  rgba(242, 92, 5, 0.08);

  /* Secondary — 네이비 (신뢰의 근거) */
  --navy:     #0F1F5C;
  --brick:    #8B3A1F;        /* 벽돌 시그니처 (제품 라인) */

  /* Text */
  --ink:      #111827;
  --ink2:     #374151;
  --muted:    #6B7280;

  /* Surface */
  --light:    #F9F7F4;        /* 베이지 톤 — 따뜻한 라이트 */
  --card:     #FFFFFF;
  --border:   #E5E7EB;

  /* Status */
  --green:    #03C75A;        /* 네이버페이/정상 */
  --red:      #DC2626;
  --gold:     #D97706;        /* 베스트/리뷰 */
}
```

### 2-2. 컬러 사용 원칙
- **오렌지(#F25C05)**: CTA, 가격 강조, 베스트 배지, "구매하기" 버튼 — 한 화면당 1~2곳 집중
- **네이비(#0F1F5C)**: 헤더, 브랜드 워드마크, 신뢰 카피 ("260만 세대" 등 숫자)
- **벽돌색(#8B3A1F)**: 벽돌 발수·백화 방지 등 벽돌 라인 제품 컬러 코드
- **베이지(--light)**: 섹션 배경 — 마트/창고가 아닌 "잘 정돈된 자재 매장" 인상
- **그린(--green)**: 결제·승인·재고있음 등 긍정 상태에만 (절대 마케팅 강조에 쓰지 않음)
- 이미지·제품 사진은 흰 배경 누끼 컷 우선, 시공 후 결과물 컷은 자연광 배경

### 2-3. 타이포그래피
- **본문 (한글)**: Pretendard Variable (300/400/500/600/700/800)
- **숫자/스펙 (영문)**: IBM Plex Sans / IBM Plex Mono — kg, ㎡, % 등 단위 강조
- **보조 (한글)**: Noto Sans KR (구버전 호환)
- 행간 1.5~1.6, 본문 14~16px, 모바일 우선
- 한자/영문 혼합 시 자간 -0.5px

### 2-4. 레이아웃 원칙
- **모바일 퍼스트**: 360~414px 기준 → 1280px까지 확장
- **상품 카드**: 정사각 썸네일 + 제목 2줄 + 가격 + 베스트/할인 배지
- **카테고리 버튼**: 좌→우 가로 스와이프 또는 9분할 그리드 (정식 카테고리 9종 기준)
- **구매 동선**: 상단 검색 → 카테고리 → 상품 카드 → 상세 → 장바구니 → 결제 (5탭 이내)
- 시공가이드 영상은 상품 상세 상단에 항상 1개 이상 노출

### 2-5. 아이콘·이미지 규칙
- 아이콘: Phosphor / Material Symbols (Outlined, 24px 기본)
- 제품 사진: 1:1 비율, 흰 배경, 그림자 약하게
- 패키지 사진: 구성품 펼쳐 보이기 (상자 + 자재 + 가이드북)
- 시공 사진: Before / After 2분할 (After 우선)

### 2-6. 금지 시각
- 어둡거나 무거운 산업용 톤 (그건 POUR공법의 영역)
- 과한 광택·금속 그라디언트
- "초강력", "역대급" 같은 자극적 캐치카피 배너
- 경쟁 자재 비교 이미지

## §3. 톤앤매너

### 3-1. 한마디로
**실용형 · 안내형 · 실행 유도형 · 친절하고 직관적**

### 3-2. 5가지 톤 원칙

| 원칙 | 써야 할 표현 | 피해야 할 표현 |
|------|-------------|---------------|
| ① 결과를 먼저 | "옥상 누수, 이 패키지 하나로 끝" | "특허 받은 슈퍼복합압축시트 적용" |
| ② 쉬운 말로 | "물 새는 옥상 / 갈라진 외벽" | "도막 들뜸 / 박락 / 백화 / 핀홀" |
| ③ 가능하다는 신호 | "처음이어도 가이드대로 따라하세요" | "전문가가 아니면 어렵습니다" |
| ④ 숫자로 안심 | "리뷰 ★4.9 / 283건" | "많은 분이 만족하셨습니다" |
| ⑤ 다음 행동 한 가지만 | "지금 장바구니 담기" | "더 알아보기 / 문의 / 상담 / 견적" 동시 노출 |

### 3-3. 카피 라이팅 톤
- **문장 길이**: 평균 15자 이내, 최대 25자
- **종결어미**: "~하세요", "~했어요", "~가능해요" (반말 금지, 하십시오체 금지)
- **숫자**: 가능하면 정수, 단위 한국어 ("3.5kg" → "3.5kg / 1세대용")
- **이모지**: 카테고리 표시 한정 (🛠 도구 / 💧 방수 / 🎨 도장) — 가격·후기에는 사용 금지
- **느낌표**: 화면당 1개 이내

### 3-4. 톤 비교 (다른 브랜드와 차별화)

| 상황 | POUR공법 | POUR솔루션 | **POUR스토어** | 그로홈 |
|------|----------|-----------|---------------|--------|
| 옥상 누수 | "POUR슬라브 듀얼강화방수공법 적용" | "현장 진단 후 맞춤 시공 제안" | **"옥상 누수 패키지 — 1박스로 5㎡"** | "주말에 베란다 직접 고쳐봐요" |
| 가격 표기 | (비공개) | (현장 견적) | **89,000원 (3.5kg)** | 89,000원 ★4.9 |
| CTA | 시방서 다운로드 | 현장진단 신청 | **장바구니 / 바로 구매** | 지금 구매 / 쿠폰 |

### 3-5. 표시광고법 준수 (POUR스토어 필수)
> 자사몰·마켓플레이스 노출 시 표시광고법 위반 가능성 있는 표현 금지

| 한번에 차단, 즉시 효과 | 케어, 저감, 억제 |
| 빠른 효과, 빠르게 | 효율적, 간편한 |
| 완벽 방수, 영구 보장 | 방수 기능, 내구성 향상 |

### 3-6. 금지어
- 견적 · 맞춤 · 진단 · VE제안 · 시방서 (B2B 영업 언어 — POUR솔루션·POUR공법 영역)

## §4. 페르소나

> POUR스토어는 단일 페르소나가 아닌 4종 페르소나가 같은 채널을 다른 동선으로 사용한다.

### 4-1. 페르소나 A — 셀프시공 입문자 (자사몰·마켓플레이스)
- **이름**: 김자영(43) / 자가 아파트 거주
- **상황**: 베란다 누수, 단지 공사 차례 대기 중. 일단 임시로라도 막고 싶음
- **검색어**: "옥상 누수 셀프", "베란다 방수제 추천"
- **결정 기준**: ① 실패해도 큰 손해 없을지 ② 시공 영상 있는지 ③ 후기 별점
- **이탈 포인트**: 전문 용어, 도구 필요량 모호, 배송 후 어떻게 쓸지 막막함
- **응답 톤**: "주말 1시간이면 충분해요. 영상대로만 하시면 돼요"
- **추천 동선**: 카테고리 → 패키지 → 가이드 영상 → 후기 → 구매

### 4-2. 페르소나 B — 소규모 시공업자 (자사몰·사방넷·B2B)
- **이름**: 박실장(51) / 1인 방수업체
- **상황**: 다음 주 현장 자재 빠르게 받아야 함. 단가 + 재고 + 배송속도가 핵심
- **검색어**: 제품명 + kg수 직접 입력 ("POUR코트재 18kg")
- **결정 기준**: ① 당일 출고 ② 묶음 단가 ③ 시방서 일치 여부 (현장 발주처 검증)
- **이탈 포인트**: 재고 미표시, 세금계산서 발행 불가, 묶음 할인 없음
- **응답 톤**: "당일 출고 가능 / 5박스 이상 단가 별도"
- **추천 동선**: 검색 → 상품 → 사업자 로그인 → 묶음 견적 → 세금계산서 발행

### 4-3. 페르소나 C — 파트너사 (자재 유통/시공 매칭)
- **이름**: 이대표(48) / 지역 자재상 운영
- **상황**: 단골 고객이 POUR 제품 찾음. 직접 결제 받지 않고 본사에 연결만
- **결정 기준**: ① 마진율 투명 ② 정산 빠름 ③ 본사 응대 친절
- **운영 모델**: **고객 정보 + 품목 입력 → POUR스토어 결제 안내 → 월말 자동 수수료 정산**
- **응답 톤**: "고객은 POUR스토어로 직접 결제, 매월 말일 등록 계좌로 수수료 입금"
- **추천 동선**: 포털 PIN 로그인 → 주문 등록 → 고객에게 결제 링크 전달 → 정산 페이지 확인

> ⚠️ **POUR공법 협력사 ≠ POUR스토어 파트너사** (자세한 구분은 §5-3 참조)

### 4-4. 페르소나 D — 매장 방문 고객 (오프라인 키오스크)
- **이름**: 정사장(60) / 동네 철물점에 들른 손님
- **상황**: 직원 설명 듣고 결정했지만 결제는 본사로
- **결정 기준**: ① 화면이 이해되는지 ② QR이 바로 떠는지 ③ 영수증 처리
- **응답 톤**: "QR을 스캔하시면 POUR스토어에서 바로 결제됩니다. 매장에서 따로 카드 받지 않습니다"
- **추천 동선**: 직원이 키오스크에 품목 입력 → QR 표시 → 고객 폰 결제 → 매장은 안내만

### 4-5. 페르소나 응답 매트릭스 (요약)

| 페르소나 | 핵심 감정 | 첫 화면 | CTA | 결제 |
|---------|---------|---------|-----|------|
| A 셀프 입문자 | "내가 할 수 있을까?" | 가이드 영상 | 장바구니 담기 | 카드/네이버페이 |
| B 소규모 시공자 | "빨리, 정확히" | 검색 + 재고 | 묶음 견적 | 세금계산서 |
| C 파트너사 | "정산만 깔끔하게" | 주문 등록 폼 | 결제 링크 발송 | 본사 결제 → 월말 정산 |
| D 매장 방문 | "복잡하지 않게" | 키오스크 카테고리 | QR 결제 | 고객 폰으로 본사 결제 |

## §5. 서비스 정책

### 5-1. 상품 정책 — 카테고리 정식 분류 9종

> ⚠️ 이 분류는 절대 단순화·통합 금지. POUR스토어 운영의 기준 분류임.

| 대분류 | 중분류 (예시) | 비고 |
|-------|-------------|-----|
| **① 주자재** | 방수재, 도장재, 코팅재, 균열보수재, 탑코트재 | 시공의 본 자재 |
| **② 부자재** | 프라이머, 실러, 퍼티, 마스킹테이프 | 주자재와 함께 사용 |
| **③ 보조자재** | 롤러, 붓, 헤라, 트레이, 비산방지막 | 도구류 |
| **④ 보강자재** | 균열보수시트, 메쉬, 구조보강 부품 | 구조 보강 부품 |
| **⑤ 안전용품** | 마스크, 장갑, 보안경, 작업복, 안전화 | PPE |
| **⑥ 패키지 상품** | "옥상 누수 패키지", "벽돌 발수 셀프 키트" | 문제 해결형 묶음 |
| **⑦ 셀프시공 키트** | 1세대용 소포장 + 가이드북 + QR | 페르소나 A 전용 |
| **⑧ B2B 대용량** | 18kg / 20kg / 1톤 단위, 묶음 단가 | 페르소나 B 전용 |
| **⑨ 도서·교육** | 시공가이드북, 영상 강의 | 부가 매출 + 신뢰 강화 |

**상품 정책 일반 원칙**
- **취급 범위**: POUR공법 시방서 등재 자재만 (외부 브랜드·OEM 자재 취급 금지)
- **신상품 등록**: 시방서 등재 → 사진 촬영 → 시공가이드 동영상 → 등록
- **단종 정책**: 시방서에서 제외된 자재는 즉시 판매 중단 + 기존 구매자 대체품 안내
- **소포장 정책**: 1세대용(1~3.5kg) ~ 단지용(20kg+) 동일 품질 원칙

### 5-2. 가격 정책 — 8단 가격 구조

> ⚠️ 가격은 채널별로 명확히 다르며, 절대 혼동·임의 적용 금지.

| 단계 | 가격 유형 | 적용 채널 | 비고 |
|------|---------|---------|------|
| **1단** | 본사 원가 | (내부) | 노출 금지 |
| **2단** | 파트너사 공급가 | POUR스토어 파트너사 (이대표 등) | 마진 정산 기준 |
| **3단** | 대형 시공사 공급가 | 사업자 로그인 후 묶음 단가 (페르소나 B 일부) | 월 1톤+ 발주 기준 |
| **4단** | B2B 도매가 | 사업자 로그인 후 단가표 (페르소나 B) | 5박스+ |
| **5단** | 온라인 위탁 공급가 | **스마트스토어** (직접 컨택 채널) | ⚠️ 11번가/오늘의집과 다름 |
| **6단** | 마켓플레이스 정상가 | **쿠팡, G마켓, 옥션** | 마켓 수수료 반영 |
| **7단** | 자사몰 정상가 | **자사몰 (pourstore.net, 카페24)** | 기준가 |
| **8단** | 자사몰 표시가 (정상 소비자가) | **11번가, 오늘의집** | 기준 정상가 그대로 |

**가격 정책 일반 원칙**
- **B2C 표시가**: 부가세 포함, 배송비 별도 표기
- **B2B 단가**: 사업자 로그인 후에만 노출, 묶음 단가표 제공
- **할인**: 정기 할인 금지, 시즌 행사·신상품 출시 한정 (최대 -15%)
- **쿠폰**: 첫 구매 / 리뷰 작성 / 묶음 구매 3종만 운영
- **채널 간 가격 정합성**: 8단계 구조에서 ±1단계 이상 가격 역전 금지 (자사몰이 마켓보다 비싸면 안 됨 등)

### 5-3. 파트너 체계 — POUR공법 협력사 vs POUR스토어 파트너사

> ⚠️ **두 체계는 완전히 독립된 트랙**. 한 회사가 양쪽 다 / 한쪽만 / 둘 다 아님 모두 가능.
> 절대 혼동·통합 금지.

| 구분 | **POUR공법 협력사** | **POUR스토어 파트너사** |
|------|-------------------|----------------------|
| 소속 브랜드 | POUR공법 | POUR스토어 |
| 영업 방식 | 입찰 / 수의계약 / 하도급 / 기술제안 (B2B) | 자재 유통 / 소매 매칭 (B2C+B2B) |
| 계약 대상 | 종합건설사, 전문건설사, 방수업체 | 자재상, 철물점, 소형 시공업체 |
| 핵심 자격 | 시공 면허, 기술 인증 | 사업자등록 + 본사 결제 시스템 동의 |
| 주요 매출원 | 시공 단가 | 제품 마진 (수수료 정산) |
| 정산 방식 | 시공 완료 후 단가 정산 | 월말 자동 수수료 입금 |
| 결제 처리 | 시공 계약별 | 본사 직결제 (파트너사 직접 수금 금지) |
| 등록 채널 | poursolution.net 시공문의 | pourstore.net 파트너 신청 |

**파트너사 응대 시 필수 확인**
1. "혹시 POUR공법 협력사 신청하시려는 건가요, POUR스토어 파트너사 신청하시려는 건가요?"
2. 두 체계 차이를 명확히 안내 후 분기 처리
3. 둘 다 신청 가능함도 함께 안내

### 5-4. 채널별 운영 정책 — 7개 온라인 채널 + 자사몰

| 채널 | 운영 책임 | 결제 처리 | 정산 | 가격 단계 |
|------|---------|----------|-----|----------|
| **자사몰 (카페24)** ※이전 중 | POUR스토어 본사 | 카드/계좌이체/네이버페이/카카오페이 | 즉시 | 7단 |
| **네이버 스마트스토어** | 본사 (직접 컨택 위탁) | 네이버 결제 | T+3 | 5단 |
| **11번가** | 본사 | 마켓 결제 | T+15 | 8단 |
| **G마켓** | 본사 | 마켓 결제 | T+10 | 6단 |
| **옥션** | 본사 | 마켓 결제 | T+10 | 6단 |
| **오늘의집** | 본사 | 마켓 결제 | T+7~14 | 8단 |
| **쿠팡** | 본사 | 마켓 결제 | T+15~30 | 6단 |
| **매장 키오스크** | 매장 운영 + 본사 결제 | **본사 직결제** (QR 발급) | 매장 수수료 월말 | 7단 |
| **파트너사 포털** | 파트너사 + 본사 결제 | **본사 직결제** | 파트너 수수료 월말 | 2단 마진 |

**통합 관리**
- **사방넷** → 7개 채널 주문/재고 자동 동기화
- **이전 진행 중**: 아임웹 (구) → 카페24 (신) — PG 등록 검토 후 동월 내 오픈 목표
- 핵심 원칙: **결제는 어떤 채널이든 본사가 직접 받는다**. 매장·파트너사는 카드/계좌를 직접 받지 않는다.

### 5-5. 배송 정책
- **출고 기준**: 평일 14시 이전 결제 → 당일 출고 / 이후 → 익영업일
- **배송사**: 일반 자재(택배, 사방넷→dada→세이고 파이프라인), 대량 자재(20kg+, 화물 별도 견적)
- **3PL**: 세이고 (부천), 원스톱물류창고 (오산)
- **배송비**: 5만원 이상 무료 / 미만 3,000원 / 도서산간 +3,000원
- **반품**: 미개봉 7일 / 개봉 후 화학자재는 반품 불가 (품질 안전 사유)
- **교환**: 오배송·파손 시 본사 부담 즉시 재발송

### 5-6. 정산 정책 (파트너사·매장 공통)
```
월말 영업일 → 매출 집계 → 수수료 자동 계산 (등급별 %)
            ↓
        익월 영업일 5일 이내 등록 계좌 자동 입금
            ↓
        정산명세서 포털에서 PDF 다운로드
```
- 등급(Tier)은 분기별 매출 기준으로 자동 갱신
- 정산 금액 이의 제기는 익월 15일까지 / 이후 확정

### 5-7. 고객 응대 정책
- **CS 창구**: 채널톡 / 자사몰 1:1 문의 / 전화
- **응답 시간**: 평일 영업시간 1시간 이내
- **시공 상담은 POUR솔루션으로**: 견적·진단 문의는 POUR스토어가 받지 않고 POUR솔루션 문의 폼으로 안내
- **하자 보증**: 자재 자체의 하자만 보증 (시공 결과는 보증하지 않음 — 시공 보증은 POUR솔루션 시공 시에만)

### 5-8. 금지 행위
| 금지 | 대안 |
|------|-----|
| 견적 금액 산출 | "정확한 시공 견적은 POUR솔루션 현장 진단 후" |
| 타사 자재 비교/추천 | POUR 브랜드 내 대안 제시 |
| 시공 결과 보증 | "자재 품질은 보증, 시공 결과는 시공 방법에 따라 다름" |
| 의약·약리 효능 표현 | "방수", "보수" 등 기능적 표현만 |
| 매장·파트너사가 카드/현금 직접 수금 | 본사 결제 링크/QR로 안내 |
| POUR공법 협력사 / POUR스토어 파트너사 혼용 | 두 체계 차이 명확히 안내 후 분기 |
| 표시광고법 위험 표현 | §3-5 안전 대체어 사용 |

## §6. 워크플로우

### 6-1. 자사몰(카페24) 구매 워크플로우
```
[고객] 사이트 방문
  → 카테고리 / 검색 / 베스트
  → 상품 상세 (가이드 영상 + 후기 + 패키지 구성)
  → 장바구니
  → 결제 (카드 / 계좌 / 네이버페이 / 카카오페이)
  → 주문 완료 → 배송 알림(SMS/카카오)
  → 수령 후 시공 → 후기 작성 (쿠폰 발급)

[본사 운영]
  사방넷 통합관리로 주문 자동 동기화
  → 세이고/원스톱 출고 → 송장 등록 → 자동 알림
```

### 6-2. 마켓플레이스 워크플로우 (7개 채널)
```
[등록] 자사몰 상품 → 사방넷 → 7개 마켓 동기화
[주문] 각 마켓 주문 → 사방넷 수집 → 본사 출고
[CS]   각 마켓 CS → 사방넷 → 채널톡 응대
[정산] 마켓별 정산 주기 (T+3~30)
```

### 6-3. 매장 키오스크 워크플로우 (오프라인)
```
[직원] 키오스크 부팅 → PIN 입력
  → 고객 응대하며 품목 선택
  → 수량 입력 → 합계 확인
  → 고객 정보(이름/연락처) 입력
  → 결제 QR 생성

[고객] 폰으로 QR 스캔
  → POUR스토어 결제 페이지 (모바일)
  → 결제 완료 → SMS 영수증 + 매장 수령 안내

[정산] 본사 ← 결제 수수금 / 매장 ← 수수료(월말)
```
> ⚠️ 매장은 절대 카드/현금을 직접 받지 않음. 키오스크는 안내·발급만 담당.

### 6-4. 파트너사 포털 워크플로우
```
[파트너사] 포털 접속 → 발급받은 PIN 로그인
  → "주문 등록" 클릭
  → 고객 정보 + 품목 + 수량 입력
  → 결제 링크/QR 생성
  → 카카오톡/문자로 고객에게 전송

[고객] 결제 링크 클릭 → POUR스토어 결제 → 주문 확정

[본사] 출고 → 송장 → 파트너사·고객 동시 알림

[정산] 매월 말일 영업일
  → 파트너사 매출 자동 집계
  → 등급별 수수료(%) 적용
  → 익월 5일 내 등록 계좌 자동 입금
  → 정산명세서 포털 게시
```

### 6-5. 자사몰 카페24 이전 워크플로우 (현재 진행 중)
```
[현재] 아임웹 운영
  → 상품 데이터 export
  → 카페24 import + 디자인 마이그레이션
  → PG 등록 검토 (진행 중)
  → 결제 모듈 테스트
  → 도메인 연결 (pourstore.net)
  → 사방넷 연동 재설정
  → 오픈 (동월 내 목표)
[병행] 아임웹 → 카페24 리다이렉트 설정 (SEO 보존)
```

### 6-6. 신상품 등록 워크플로우
```
1. 시방서 등재 확인 (POUR공법 공식 문서)
2. 사진 촬영 (단품 / 구성품 / 시공 전후)
3. 시공가이드 동영상 1편 (60~120초)
4. 상세페이지 작성 (구조: 문제 → 해결 → 사용법 → 주의 → 후기)
5. 표시광고법 컴플라이언스 체크 (§3-5 기준)
6. 가격·재고·배송 설정 (8단 구조 적용)
7. 자사몰 등록 → 사방넷 → 7개 마켓 동기화
8. 관리센터에 신상품 알림 (파트너사·매장에 발송)
```

### 6-7. 패키지 기획 워크플로우
```
"문제"에서 출발한다 (제품에서 출발 금지)
  예) "옥상 5㎡ 누수" 패키지
       = POUR코트재 1통 + 슈퍼복합압축시트 1롤 + 롤러/장갑/마스크
       + 시공가이드북 + 영상 QR

가격 = 단품 합계 -10% 이내 (할인 미끼 금지, 묶음 편의 가격)
구성 = 사진 한 장에 모두 보이게
대상 = 페르소나 A(셀프 입문자) 우선
```

### 6-8. 탑코트재 라인 워크플로우 (신제품 라인 — 진행 중)
```
[기획] 탑코트재 9 SKU (벽돌 발수 / 아스팔트슁글 / 외벽 / 드라이비트 /
        벽돌 백화 / 목재 방부 / 철재 방청 / 미장 마감 / 패널·철골 방청)
        → 모두 4kg 단위 (1세대용 소포장)

[콘텐츠] Excel v12 기준 (특징 4+개/SKU, 표시광고법 검토 완료)
      → 컴플라이언스 체크 (강력한·한번에·빠른 등 금지어 제거)
      → DIY 친화 표현으로 재작성 (전문용어 → 평어)

[디자인] Figma 라벨 디자인 (벽돌 발수부터 시작)
      → html.to.design 플러그인 활용
      → 라벨 전면 한 줄 카피 결정 ("한번에 차단" 대체)

[등록] Excel → Figma 라벨 → 인쇄 → 사진 → 자사몰 → 7개 마켓
```

### 6-9. CS · 환불 워크플로우
```
[접수] 채널톡 / 문의 폼 / 전화
[1차] 1시간 내 회신 → 사진 요청 (오배송/파손)
[판정]
  - 자재 자체 하자 → 즉시 재발송 (본사 부담)
  - 시공 결과 불만 → POUR솔루션 진단 안내 (자재 환불 불가)
  - 단순 변심 (미개봉 7일) → 환불 (배송비 고객 부담)
  - 화학자재 개봉 후 → 환불 불가 (품질·안전)
[종결] 처리 결과 SMS + 만족도 1문항
```

## §7. 응답 규칙 (AI 어시스턴트용)

### 7-1. 답변 4단계 구조
```
[1단계] 문제 공감 (1문장)
   "옥상에 물이 새는 상황이군요. 면적이 어느 정도일까요?"

[2단계] 해결 옵션 (1~2가지, 추천 명확히)
   "5㎡ 이내면 [옥상 누수 패키지 1박스]면 충분해요."
   "20㎡ 이상이면 [POUR코트재 18kg + 시트] 따로 구매가 경제적이에요."

[3단계] 안심 메시지 (가이드/후기/배송)
   "포함된 가이드 영상대로 따라하시면 1시간 안에 끝나요. 후기 ★4.8/167건"

[4단계] 다음 행동 1가지
   "[지금 장바구니 담기]"  ← 한 가지만 제시
```

### 7-2. 페르소나 자동 추정
- 사업자등록번호/회사 언급 → **B (소규모 시공자)** 톤으로 전환
- 자재상·철물점·정산·수수료 언급 → **C (POUR스토어 파트너사)** → 포털 PIN 안내
- 매장·키오스크·QR 언급 → **D (매장 방문)** → 본사 결제 원칙 안내
- 입찰·시공 면허·기술제안 언급 → **POUR공법 협력사 신청자** → POUR공법 채널로 안내 (POUR스토어 파트너사와 다름을 명시)
- 그 외 → 기본 **A (셀프 입문자)** 톤

### 7-3. 답변 금지 사항
1. 시공 견적 금액 제시 — "정확한 시공 견적은 POUR솔루션 현장 진단 후"
2. 결제 우회 안내 — 매장·파트너사에 카드 받으라는 안내 절대 금지
3. 타사 제품 비교/대체 추천
4. 의학적·약리적 효능 표현
5. 보증 기간 임의 약속 — 시방서·상품 상세에 표기된 범위만 인용
6. 표시광고법 위험 표현 (§3-5)
7. POUR공법 협력사와 POUR스토어 파트너사 혼용 안내

### 7-4. 시공 상담으로 전환해야 하는 신호
다음 키워드 등장 시 즉시 POUR솔루션 문의 폼으로 안내한다:
- 단지·아파트·건물 전체·옥상 전체
- 견적·계약·VE·입찰
- 하자보수·민원·관리소장
- 안내 문구: *"규모가 큰 시공은 자재 구매보다 현장 진단이 먼저예요. POUR솔루션 무료 진단을 신청해드릴까요?"*

### 7-5. 즉석 QR 안내
- 상품 결제 QR (키오스크/파트너사)
- 가이드 영상 QR (상품 카드)
- 후기 작성 QR (배송 완료 시)

## §11. 체크리스트 (현장 운영용 퀵)

### 상품 등록 전

- [ ] 카테고리 9종 중 어디에 속하는가 (단순화 금지)
- [ ] 누끼 사진(흰배경) 1장 이상
- [ ] 시공가이드 영상 60~120초
- [ ] 단품/패키지 결정
- [ ] 소포장 단위 결정
- [ ] 가격 8단 구조 중 채널별 단가 결정
- [ ] 사방넷 카테고리 매핑

### 카피 작성 전
- [ ] 첫 줄에 결과(=문제 해결)가 들어 있는가
- [ ] 페르소나 A 기준으로 읽힌다
- [ ] 종결어미 "~하세요/했어요"
- [ ] 견적·진단·시방 같은 B2B 단어 없음
- [ ] CTA 1개
- [ ] **표시광고법 위험 표현 0건** (강력한, 한번에, 빠른, 완벽 등)

### 채널 운영 전
- [ ] 결제는 본사 직결제인가 (매장/파트너사가 직접 받지 않는가)
- [ ] 정산 등급·수수료율 확정
- [ ] 포털 PIN 발급 채널 정리
- [ ] 7개 채널별 가격 단계 정합성 확인 (역전 없음)

### 파트너사 응대 전
- [ ] POUR공법 협력사 / POUR스토어 파트너사 어느 쪽인지 확인
- [ ] 두 체계 차이 안내 후 분기 처리
- [ ] 둘 다 신청 가능함도 함께 안내
